  <b>Live E-Commerce Data Monitoring System</b><br><br>

  This project is an end-to-end real-time data monitoring solution that extracts product information from a live API, automates data processing, and delivers meaningful business insights through email automation and an interactive Power BI dashboard.<br><br>

  <b>üöÄ Project Overview</b><br><br>

  The system fetches real-time e-commerce product data (price, rating, stock, etc.) using an automated HTTP GET request. The raw data is then cleaned, filtered, and transformed using JavaScript, ensuring the dataset is accurate and analysis-ready.<br><br>

  After processing, the data is automatically stored in a structured spreadsheet, which acts as the single source of truth for both the dashboard and the automated insight email. This workflow runs automatically, ensuring the dataset refreshes with new information every time the flow executes.<br><br>
